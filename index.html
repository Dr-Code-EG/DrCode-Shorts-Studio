ملفات الأداة

DrCode-Shorts-Studio/
main.py الملف الرئيسي - واجهة المستخدم
modules/
config_manager.py إدارة الإعدادات و API Keys
gemini_handler.py التواصل مع Gemini AI
tts_handler.py تحويل النص إلى صوت
pexels_handler.py البحث عن الصور
video_maker.py تركيب الفيديو بـ MoviePy
youtube_uploader.py النشر على YouTube
progress_tracker.py عرض التقدم
output/ مجلد حفظ الفيديوهات
temp/ ملفات مؤقتة أثناء التشغيل
assets/ الخطوط والصور الافتراضية
requirements.txt المكتبات المطلوبة
README.md توثيق المشروع
.gitignore
config.json ملف الإعدادات (يُنشأ تلقائياً)

نقاط مهمة

  • التثبيت بأمر واحد
  • الإعداد تلقائي في أول تشغيل
  • دعم مفاتيح Gemini متعددة
  • تبديل تلقائي عند انتهاء الحد
  • حفظ الإعدادات في config.json
  • الفيديوهات تُحفظ في مجلد output/

التثبيت

كيفية التثبيت والتشغيل

1

استنساخ المستودع

Terminal 
git clone https://github.com/drcode-ai/drcode-shorts-studio.git
cd drcode-shorts-studio
2

تثبيت المكتبات

Terminal 
pip install -r requirements.txt
3

تشغيل الأداة

Terminal 
python main.py

في أول تشغيل، ستُطلب منك إدخال API Keys (Gemini, Pexels, TTS) وستُحفظ تلقائياً
